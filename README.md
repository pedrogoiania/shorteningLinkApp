## Requirements

- node >= 20.19
- npm >= 10.8
- yarn >= 1.22
- xcode >= 16.2
- android studio >= 2025.1.1

## Get started

1. Install dependencies

   ```bash
   yarn install
   ```

2. Start the app

   ```bash
   yarn start --reset-cache
   ```

3. Then press `i` for iOS, `a` for Android, `w` for web, to load the app.

## To run the tests

1. Just to run the tests

   ```bash
   yarn test
   ```

2. If you want to see the coverage

   ```bash
   yarn test:coverage
   ```
